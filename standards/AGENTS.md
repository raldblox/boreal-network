# Standards Namespace Guidelines

## Scope

This `AGENTS.md` applies to `C:\Users\raldb\boreal-network\standards` unless a deeper child workspace adds a more specific file.

## Rules

- `standards/*` is for protocol profiles, implementation standards, and compatibility notes.
- Standards may refine implementation guidance.
- Standards may not override canonical object meaning, lifecycle meaning, or payment meaning.
- If a standard needs research on scope, compatibility, protocol shape, or adapter boundary, do that alignment first and record it before turning the standard into a durable file set.
