# Papers

This folder holds research manuscripts, paper drafts, figures, and venue-facing notes derived from Boreal canon.

These documents are secondary to root canon.
They may explain, evaluate, or package Boreal ideas for publication.
They must not redefine canonical objects, lifecycle meaning, or contract semantics.

If a paper draft introduces a new semantic claim, update canon first or in the same patch.

Recommended manuscript layout:

- one folder per paper slug
- `README.md` for paper scope and status
- `abstract.md` for short-form draft text
- `outline.md` for section-by-section narrative
- `paper.tex` for two-column or venue-specific source
- `references.bib` for bibliography
- `figures/` for diagrams and exported images
- `notes.md` for benchmark, artifact, and submission notes

This folder is for publication work, not runnable product code.
